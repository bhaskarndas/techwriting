ColorAppIconColorString
[MaxLen: 7]
[Regex: #([A-Fa-f0-9]{6})]		Specify the web color used to provide the icon background when the entity icon is displayed. The web color must be specified in web hex format and must begin with a leading # sign and contain 6 additional characters consisting of the letters A-F and the numbers 0-9. For example, the code ‘#00204F’ specifies the color we use for the main menu background. You can select a color from the color picker or enter one directly.
