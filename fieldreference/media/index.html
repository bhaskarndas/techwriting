Media

The Media entity stores links to media such as PDFs, images, and spreadsheets uploaded by employees or clients. It is a system table and cannot be edited by users.

Field LabelField NameField TypeDescription
Media IdMediaIdGuid
[Key]
[Required]
[ReadOnly]
Blob NameBlobNameStringMedia such as images, documents and other files are stored as BLOBs (Binary Large Objects) in Azure Cloud Storage.

Mime TypeMimeTypeStringA media type (also known as a Multipurpose Internet Mail Extensions or MIME type) is a standard that indicates the nature and format of a document, file, or assortment of bytes, eg ‘application/vnd.ms-excel’.
File SizeFileSizeInt64Size of the media file that has been uploaded in Bytes.
